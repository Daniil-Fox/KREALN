сопровождение прохождения государственной экспертизы

Этапы проведения модернизации очистных сооружений:
  • 1. Обследование

    В ходе обследования очистных сооружений выясняется их фактическое техническое состояние. Заказчик на выходе получает отчет о проведенном обследовании.

  • 2. Разработка вариантов технической концепции модернизации

    Разрабатываются подходящие для ситуации на объекте варианты технической реализации модернизации комплекса очистных сооружений. Заказчик на выходе получает варианты модернизации очистных сооружений.

  • 3. Расчет ТЭО для каждого из вариантов модернизации

    Выполняется оценка экономической целесообразности для выбранных заказчиком вариантов модернизации очистных сооружений. Заказчик на выходе получает утверждение одного наиболее приемлемого варианта для проведения модернизации по цене, срокам, видам работ, размещению, типам оборудования, окупаемости.

  • 4. Разработка проектной документации «П» и «Р»

    Выполняется проектирование всех необходимых разделов ПСД для модернизации очистных сооружений. Заказчик на выходе получает согласованную и прошедшую экспертизу ПСД.

  • 5. Поставка оборудования

    Проводится поставка оборудования. Заказчик на выходе получает закупленное и готовое к монтажу оборудование.

  • 6. Строительно-монтажные работы

    Выполняются строительно-монтажные работы на объекте. Заказчик на выходе получает законченный монтаж комплекса очистных сооружений.

  • 7. Пуско-наладка

    Проводится пуско-наладка оборудования и настройка технологических режимов работы до выхода показателей на норматив сброса. Заказчик на выходе получает принятые в эксплуатацию очистные сооружения.
